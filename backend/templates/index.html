Sobre FreshGuard

FreshGuard es un sistema IoT autonomo desarrollado en la Universidad Nacional Mayor de San Marcos que monitorea en tiempo real las condiciones de conservacion de frutas almacenadas.

El sistema integra sensores de gases, temperatura y humedad conectados a un microcontrolador ESP32, transmitiendo datos cada 3 segundos a un servidor en la nube mediante protocolo HTTP REST con autenticacion por API Key.

La toma de decisiones autonoma se implementa mediante el paradigma MAPE-K de Autonomic Computing, clasificando el estado de conservacion en cuatro etapas: Verde, Maduro, Sobremaduro y Podrido.
