Sobre FreshGuard

FreshGuard es un sistema IoT autónomo desarrollado en la Universidad Nacional Mayor de San Marcos que monitorea en tiempo real las condiciones de conservación de frutas almacenadas.

El sistema integra sensores de gases, temperatura y humedad conectados a un microcontrolador ESP32, transmitiendo datos cada 3 segundos a un servidor en la nube mediante protocolo HTTP REST con autenticación por API Key.

La toma de decisiones autónoma se implementa mediante el paradigma MAPE-K de Autonomic Computing, clasificando el estado de conservación en cuatro etapas: Verde, Maduro, Sobremaduro y Podrido.
